PSuccE: Detecting Succinylation Sites from Protein Sequences using Ensemble Support Vector Machine.

These are some source codes and database for paper 'PSuccE'.

AAC.cpp: source code for feature Amino Acid Composition;
BE.cpp: source code for feature Binary Encoding;
aaindex.cpp: source code for feature Physicochemical Property;
GM.cpp: source code for feature Grey Pseudo Amino Acid Composition;
IG1~IG10: the number of dimension selected from whole feature space;
model1~model10: models in the first layer SVM;
model: model of the second layer SVM;
Database.rar: dataset of training data and independent test data.
